Getting started

Three steps to go.

01 — 3DS

Install the plugin

Copy the .3gx plugin to your SD card under /luma/plugins/ and enable it via the Luma plugin loader.

02 — PC

Run the client

Download 3DS-DiscordRPC.exe and launch it. Enter your 3DS's local IP when prompted — that's it.

03 — Discord

Play & show off

Launch any game on your 3DS. Your Discord status updates automatically with the game name and icon.

Requirements

What you need.

🔧 Custom firmware (Luma3DS)
💻 Windows 10/11 PC
🌐 3DS & PC on same network
💬 Discord desktop app
